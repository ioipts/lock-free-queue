# Lock Free Queue

Thread safe Circular queue (Ring queue).
It uses atomic operation to be lock-free.

## Feature
Written in C++11.  
Support multi-platform.  
Support both 32-bit and 64-bit.  

## How to use
Just copy axisqueue.h and include in the project.  
Enjoy!  

## How to run the sample
cd sample  
make  
./testqueue
